About Author

ITC

An ITC course portfolio collection by Shane Mansueto. Listing submitted assignments from the 1st semester of s.y. 2024-2025. Containing portfolios#1-3 (Midterms) and portfolios#4-6 (finals). A learning experience on HTML tags, CSS styles, and web design.

"Purity"

Pure custom HTML and CSS, no external libraries, files nor reasources.

Challenges

Impact vs usability. Interaction or ease-of-use. Implementing features that would amaze users or maintaining minimal abilities to have the least stressful and easiest UI

Developing for the user or for myself. Creating a better experience in viewing the site, or a better experience making the site. Allocating time to have a better development enviroment, taking time off generating the actual contens of a page.

Untainted is based on the concept of develped only using native HTML and CSS files, and functionality. If it's not a HTML/CSS script then it doesn't won't exist in my website. The inclusion of images, audio, videos, programming-languages, and other file types would not be present in my site.

A lot of documentation is needed to be read to form functions and design. There are numerous methods to perform an action in HTML and CSS. In reference the number of different ways to select in CSS is astounding. It is difficult to know whether or not a simple method exists for a certain task or not. As such this particular interactive block which I had improvised with CSS, I later found out can be done using only HTML.
